### 推送类型说明

悦航益平台推送 `POST` JSON 到回调地址，`dataType` 字段区分：

| dataType          | 说明             | 落库表                              |
| ----------------- | ---------------- | ----------------------------------- |
| `UploadLog`       | 录音文件上传完成 | `audio_record`                      |
| `AudioText`       | 语音转写文本     | `audio_record` (更新)               |
| `GpsLog`          | GPS定位数据      | `device_gps_log`                    |
| `DeviceRecording` | 录音开始/结束    | `device_event_log` + `audio_record` |
| `ConnectLog`      | 设备连接/断开    | `device_event_log`                  |
| `HeartbeatLog`    | 设备心跳         | `device_event_log`                  |
| `ControlLog`      | 指令执行结果     | `device_event_log`                  |
| `DebugLog`        | 调试信息         | `device_event_log`                  |
