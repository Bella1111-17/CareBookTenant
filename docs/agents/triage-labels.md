# Triage 标签

五个标准分类标签，对应 GitHub Issues（及 PR）上的 label 名称。

| 角色 | 标签 | 含义 |
|------|------|------|
| needs-triage | `needs-triage` | 需要维护者评估 |
| needs-info | `needs-info` | 等待报告者补充信息 |
| ready-for-agent | `ready-for-agent` | 已完全明确，可交由 Agent 处理 |
| ready-for-human | `ready-for-human` | 需要人工实现 |
| wontfix | `wontfix` | 不会处理 |

---

## 使用方式

这些标签由 `/triage` skill 自动应用，不要手动打标。

## 自定义覆盖

无覆盖。直接使用标准标签名称。
